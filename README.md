# infinitecodes-solutions.de

Personal portfolio website for **infinitecodes-solutions** — built with React, hosted on a self-managed Linux server with Nginx and HTTPS via Let's Encrypt.

## 🚀 Tech Stack

- **Frontend:** React (Create React App)
- **Styling:** Inline styles + CSS variables (dark/light mode)
- **Fonts:** Inter — self-hosted (GDPR compliant, no Google Fonts)
- **Server:** Nginx on Fedora/RHEL
- **SSL:** Let's Encrypt via Certbot
- **Deployment:** Manual build & deploy via SCP

## ✨ Features

- 🌍 German / English language toggle
- 🌙 Dark / Light mode (persisted via localStorage)
- 📱 Responsive design with mobile burger menu
- ⚡ Optimized production build (minified, hashed assets, gzip)
- 🔒 GDPR compliant — no external trackers, no cookie banner needed
- 🗂️ Sections: Hero, About, Projects, Skills, Contact, Footer
- 📄 Impressum & Datenschutz pages
